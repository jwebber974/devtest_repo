IntelliGuard Programming Test

The test consists of the following tasks:

  1. Add the ability to add, update and list all users using the user service.
  2. Clone the user models before returning them to the UI.
  3. Add the sql to the sql files in Sql/Users to match the User model.
  4. Add a page with a single textbox and button that checks if the contents of the textbox is a palindrome. Don't look this up, try it yourself.
  5. Add the interfaces needed to add dependency injection.
  6. Add dependency injection to the project and inject the appropriate classes. Add the user service as a Singleton and the palindrome service as Scoped.
  7. Add a unit test(s) to the palindrome service.
